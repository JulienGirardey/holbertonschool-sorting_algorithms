# Holberton School - Sorting Algorithms

Ce dépôt contient des implémentations de différents algorithmes de tri en langage C, réalisés dans le cadre du cursus Holberton School.

## Objectif

Comprendre et implémenter les principaux algorithmes de tri :
- Tri à bulles (Bubble sort)
- Tri par insertion (Insertion sort)
- Tri par sélection (Selection sort)
- Tri rapide (Quick sort)

## Fichiers

- `0-bubble_sort.c` : Tri à bulles
- `1-insertion_sort_list.c` : Tri par insertion sur liste
- `2-selection_sort.c` : Tri par sélection
- `3-quick_sort.c` : Tri rapide
- `print_array.c`, `print_list.c` : Fonctions utilitaires d'affichage
- `sort.h` : Fichier d'en-tête
- `*-O` : Fichiers d'analyse de complexité

## Compilation

Compiler avec gcc :
```bash
gcc -Wall -Werror -Wextra -pedantic <fichier>.c print_array.c print_list.c -o <executable>
```

## Utilisation

Exécuter l'exécutable généré pour tester chaque algorithme.
